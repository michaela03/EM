FeedEk Examples

Basic Example

Code 
$('#divFeed').FeedEk({
    FeedUrl: 'https://jquery-plugins.net/rss',
    MaxCount: 2
});

Description Character Limit Example

Code 
$('#divFeed').FeedEk({
    FeedUrl: 'https://jquery-plugins.net/rss',
    MaxCount: 3,
    ShowDesc: true,
    ShowPubDate: false,
    DescCharacterLimit: 100
});

Multiple Feed Url Example

Code 
$('#divFeed').FeedEk({
    FeedUrl: ['http://feeds.bbci.co.uk/news/rss.xml', 
              'https://www.espn.com/espn/rss/news'],
    MaxCount: 4
});
